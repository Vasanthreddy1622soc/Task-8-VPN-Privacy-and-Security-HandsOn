## üìù Task Overview

### Objectives

- Demonstrate IP masking and encrypted network traffic

### Steps Performed
1. Selected ProtonVPN as a reputable free VPN provider
2. Created account, downloaded, and installed VPN client
3. Connected to the fastest free server (Singapore)
4. Verified IP change and encrypted browsing using HTTPS checks
5. Disconnected VPN to compare behavior and network speed
6. Documented observations and screenshots as proof of execution
